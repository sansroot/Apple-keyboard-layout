## Installation

1. Copy text below and paste into your terminal
   
   ```
   git clone https://github.com/sansroot/Apple-keyboard-layout
   cd Apple-keyboard-layout
   sudo chmod -R 777 *
   bash ./installer.sh
   ```
2. Done!
